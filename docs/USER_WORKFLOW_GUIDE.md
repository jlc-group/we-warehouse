## 1. Desktop: หน้าภาพรวม

**เข้าถึง:** เปิดเว็บ → Login → หน้าแรกคือ Dashboard

### ขั้นตอน
1. เลือก **Warehouse** ที่ต้องการดู (dropdown ด้านบน)
2. เลือกโหมดดู:
   - **โหมดใช้งาน** — แสดงแผนผังคลัง + ข้อมูลสินค้า
   - **โหมดมอนิเตอร์** — แสดง dashboard วิเคราะห์
3. ดูสถิติด้านขวา:
   - จำนวนรายการทั้งหมด / ตำแหน่งที่ใช้
   - แยกตาม FG (สินค้าสำเร็จรูป) / PK (วัสดุบรรจุภัณฑ์)
4. **การดำเนินการด่วน** (แถบขวา):
   - `เพิ่มสินค้า` → เปิด modal เพิ่มสินค้าใหม่
   - `ย้ายสินค้า` → ไปหน้าทำงานเป็นชุด
   - `ส่งออก` → เปิด dialog ส่งออกทีละรายการ
   - `สแกน QR Code` → เปิดกล้องสแกน
   - `ส่งออกหลาย` → เปิด modal ส่งออกหลายรายการ (ปุ่มสีเขียว)

## 2. Desktop: รายการสินค้า

**เข้าถึง:** Sidebar → คลังสินค้า → รายการสินค้า

### ขั้นตอน
1. ดูตารางสินค้าทั้งหมดในคลัง
2. **ค้นหา:** พิมพ์ชื่อสินค้า, SKU, หรือ Location ในช่องค้นหา
3. **กรอง:** เลือก Lot, รหัสสินค้า, ประเภท, สถานะ
4. **จัดเรียง:** คลิกหัวคอลัมน์เพื่อเรียงลำดับ
5. **Export:** กดปุ่ม Export Excel หรือ Export PDF ด้านบน
6. แต่ละรายการแสดง:
   - ชื่อสินค้า + SKU + ประเภท (FG/PK)
   - ตำแหน่ง (Location)
   - จำนวนแยกหน่วย (ลัง / กล่อง / ชิ้น)
   - รวม (ชิ้น) — คำนวณจากอัตราแปลง

## 3. Desktop: จัดการสินค้า

**เข้าถึง:** Sidebar → คลังสินค้า → จัดการสินค้า

### ขั้นตอน
1. เลือก sub-tab:
   - **รายการสต็อก** — ตารางสินค้าพร้อม actions
   - **สรุปสินค้า** — สรุปรวมตาม SKU
   - **จัดการข้อมูลสินค้า** — เพิ่ม/แก้ไขข้อมูลสินค้าหลัก
2. **เพิ่มสินค้าใหม่:**
   - กด `+ เพิ่มสินค้า`
   - กรอก: SKU, ชื่อสินค้า, ประเภท (FG/PK), หมวดหมู่, แบรนด์
   - กด `บันทึก`
3. **แก้ไขสินค้า:**
   - กดไอคอนแก้ไขที่แถวสินค้า
   - แก้ข้อมูล → กด `บันทึก`

## 4. Desktop: แผนผังคลัง (Shelf Grid)

**เข้าถึง:** Sidebar → คลังสินค้า → แผนผังคลัง

### ขั้นตอน
1. ดูแผนผังเป็น Grid — แถว A-Z, ชั้น 1-4, ตำแหน่ง 1-20
2. แต่ละช่องแสดง:
   - รหัสตำแหน่ง (เช่น A3/2)
   - จำนวนรายการสินค้า
   - จำนวนรวม (ชิ้น)
   - สีแสดงสถานะ (เขียว = มีสินค้า, เทา = ว่าง)
3. **คลิกที่ช่อง** → เปิดหน้ารายละเอียด Location
   - แสดงสินค้าทั้งหมดที่ตำแหน่งนั้น
   - ปุ่ม: ย้ายสินค้า, เพิ่มสินค้า, ดูประวัติ
4. **ค้นหา:** พิมพ์ชื่อสินค้าหรือรหัส → Grid highlight ตำแหน่งที่พบ
5. **กรองตามประเภท:** กดปุ่ม ทั้งหมด / FG / PK / RM

## 5. Desktop: จัดการ Location

**เข้าถึง:** Sidebar → คลังสินค้า → จัดการ Location

### ขั้นตอน
1. ดูรายการ Location ทั้งหมดในระบบ
2. **เพิ่ม Location ใหม่:**
   - กด `+ เพิ่ม Location`
   - กรอก: รหัส (เช่น A1/1), แถว, ชั้น, ตำแหน่ง, ประเภท (shelf/floor/special)
   - กด `บันทึก`
3. **แก้ไข:** กดไอคอนแก้ไข → เปลี่ยนข้อมูล → บันทึก
4. **ลบ:** กดไอคอนลบ → ยืนยัน (soft delete)

## 6. Desktop: ส่งออกหลายรายการ (Bulk Export)

**เข้าถึง:** หน้าภาพรวม → ปุ่มสีเขียว `ส่งออกหลาย` (แถบขวา)

### ขั้นตอน (3 Steps)

**Step 1: เลือกสินค้า**
1. ดูรายการสินค้าในคลังทั้งหมด (ซ้าย)
2. ค้นหาสินค้าด้วยชื่อ, SKU, หรือ Location
3. กรองตามประเภท: ทั้งหมด / FG / PK
4. กดปุ่ม `+` ที่แถวสินค้าเพื่อเลือก → สินค้าย้ายไปฝั่งขวา
5. ตรวจสอบรายการที่เลือก (ขวา)
6. กด `ต่อไป: แบ่งให้ลูกค้า`

**Step 2: แบ่งให้ลูกค้า**
1. สำหรับแต่ละสินค้าที่เลือก:
   - กด `+ เพิ่มลูกค้า`
   - เลือกลูกค้า (dropdown)
   - ใส่จำนวน (ลัง / กล่อง / ชิ้น)
   - สามารถเพิ่มหลายลูกค้าต่อสินค้า
2. ตรวจสอบว่าจำนวนรวมไม่เกินสต็อกที่มี
3. กด `ต่อไป: สรุปและยืนยัน`

**Step 3: สรุปและยืนยัน**
1. ดูสรุปการส่งออกทั้งหมด:
   - สินค้า, ลูกค้า, จำนวน
2. ใส่เลข PO/Invoice (ถ้ามี)
3. กด `ยืนยันการส่งออก`
4. ระบบบันทึก → หักสต็อก → บันทึกประวัติ

## 7. Desktop: QR Code Management

**เข้าถึง:** Sidebar → เครื่องมือ → QR Code

### สร้าง QR เดี่ยว
1. ใส่รหัส Location (เช่น A3/2) ในช่อง
2. กด `สร้าง QR`
3. ระบบ generate QR Code ที่ encode URL: `/mobile/location/A3-2`
4. กด `Download` → ดาวน์โหลด PNG
5. กด `Print` → พิมพ์ QR

### สร้าง QR หลายตำแหน่ง (Bulk)
1. เลือกช่วง Location เริ่มต้น - สิ้นสุด
2. กด `สร้างทั้งหมด`
3. ระบบ generate QR ทุกตำแหน่งในช่วง
4. กด `ดาวน์โหลดทั้งหมด`

### จัดการ QR ที่มี
1. ดูรายการ QR ที่สร้างแล้ว
2. ค้นหาตาม Location
3. กด Download / Print / Delete ต่อ QR

### วิธีใช้ QR
- พิมพ์ QR → ติดที่ชั้นวาง/ตำแหน่งนั้น
- พนักงานใช้มือถือสแกน → เปิดหน้าจอทำงานทันที

## 8. Desktop: Staging (พักสินค้า)

**เข้าถึง:** Sidebar → การส่งออก → พักสินค้า (Staging)

### ขั้นตอน
1. ดูรายการที่รออยู่ใน staging queue
2. แต่ละรายการแสดง:
   - ประเภท (รับเข้า / ย้าย / เบิก)
   - สินค้า + จำนวน
   - จาก Location → ไป Location
   - ผู้ทำรายการ + เวลา
   - สถานะ (pending / confirmed / cancelled)
3. **ยืนยัน:** กดปุ่ม `ยืนยัน` → ระบบดำเนินการจริง (หัก/เพิ่มสต็อก)
4. **ยกเลิก:** กดปุ่ม `ยกเลิก` → ลบออกจาก queue
5. **ยืนยันทั้งหมด:** กด batch confirm

## 14. Mobile: สแกน QR ที่ Location

**เข้าถึง:** สแกน QR Code ที่ติดที่ชั้นวาง

### ขั้นตอน
1. เปิดกล้องมือถือ หรือใช้เครื่องสแกนมือถือ
2. สแกน QR Code ที่ติดอยู่ที่ตำแหน่ง
3. ระบบเปิดหน้า **Location Action** โดยอัตโนมัติ
4. หน้าจอแสดง:
   - **ชื่อ Location** (เช่น A3/2) + Zone + ชั้น + ระดับ
   - **สินค้าปัจจุบัน** — รายการสินค้าที่อยู่ ณ ตำแหน่งนี้ พร้อมจำนวน
   - **4 ปุ่ม Action:**

| ปุ่ม | สี | ทำอะไร |
|---|---|---|
| 📥 รับเข้า | เขียว | เปิดหน้ารับสินค้า (pre-fill location นี้) |
| 📤 ส่งออก | แดง | เปิดหน้าเบิกจ่าย (pre-fill location นี้) |
| 🔄 ย้ายสินค้า | น้ำเงิน | เปิดหน้าย้าย (pre-fill ต้นทาง location นี้) |
| 📋 นับสต็อก | ม่วง | เปิดหน้านับสต็อก (pre-fill location นี้) |

5. **ประวัติล่าสุด** — แสดง 10 กิจกรรมล่าสุดที่เกิดขึ้นที่ตำแหน่งนี้
   - ใคร ทำอะไร เมื่อไหร่ จำนวนเท่าไหร่
6. **ระบบบันทึก "สแกน" อัตโนมัติ** เป็น activity log

## 15. Mobile: รับสินค้า (Receive)

**เข้าถึง:** หน้าแรก → `รับสินค้า` หรือ สแกน QR → `รับเข้า`

### ขั้นตอน

**Step 1: สแกน PO**
1. สแกน barcode ใบ PO หรือพิมพ์เลข PO
2. ระบบค้นหาและแสดงรายการสินค้าใน PO

**Step 2: รับสินค้าทีละรายการ**
1. เลือกสินค้าจากรายการ PO (หรือสแกน SKU)
2. ใส่ **Location ที่จะวาง** (หรือ pre-fill จาก QR scan)
3. ใส่ **จำนวนที่รับจริง** (ลัง / กล่อง / ชิ้น)
4. กด `ยืนยันรับ`
5. รายการเข้า **staging queue** (รอ confirm ที่ desktop)
6. แสดง animation สำเร็จ ✓

**ทำซ้ำ** จนครบทุกรายการใน PO

## 16. Mobile: เบิกจ่าย (Pick)

**เข้าถึง:** หน้าแรก → `เบิกจ่าย` หรือ สแกน QR → `ส่งออก`

### ขั้นตอน

**Step 1: สแกน Order**
1. สแกน barcode ใบ order หรือพิมพ์เลข order
2. ระบบแสดงรายการสินค้าที่ต้องหยิบ (เรียงตาม location)

**Step 2: หยิบสินค้าทีละรายการ**
1. ดูรายการสินค้าถัดไป:
   - ชื่อสินค้า + SKU
   - **ตำแหน่งที่ต้องไปหยิบ**
   - จำนวนที่ต้องหยิบ
2. เดินไปที่ตำแหน่ง → **สแกน QR ที่ชั้น** เพื่อยืนยันว่าอยู่ถูกที่
3. หยิบสินค้า → ใส่ **จำนวนที่หยิบจริง**
4. กด `ยืนยัน`
5. ระบบแสดงรายการถัดไป

**Step 3: เสร็จสิ้น**
1. หยิบครบทุกรายการ → แสดง progress 100%
2. กด `ส่งทั้งหมด`
3. รายการเข้า **staging queue**

---

## 17. Mobile: ย้ายสินค้า (Move)

**เข้าถึง:** หน้าแรก → `ย้ายสินค้า` หรือ สแกน QR → `ย้ายสินค้า`

### ขั้นตอน

1. **เลือกสินค้าต้นทาง:**
   - สแกน Location ต้นทาง (หรือ pre-fill จาก QR)
   - เลือกสินค้าจากรายการที่แสดง
2. **เลือก Location ปลายทาง:**
   - สแกน QR ที่ Location ปลายทาง
   - หรือพิมพ์รหัส Location (เช่น B5/3)
3. **ใส่จำนวน:**
   - จำนวนที่ต้องการย้าย (ชิ้น)
   - ระบบแสดง: ต้นทาง → ปลายทาง (visual flow)
4. **กด `ยืนยันย้าย`**
5. รายการเข้า **staging queue**

---

## 18. Mobile: นับสต็อก (Count)

**เข้าถึง:** หน้าแรก → `นับสต็อก` หรือ สแกน QR → `นับสต็อก`

### ขั้นตอน

**Step 1: เลือก Location**
1. สแกน QR ที่ Location (หรือ pre-fill จาก QR)
2. ระบบแสดงรายการสินค้าที่ควรอยู่ที่ตำแหน่งนี้

**Step 2: นับทีละรายการ**
1. สำหรับแต่ละสินค้า:
   - ดูจำนวนที่ระบบบันทึกไว้ (expected)
   - **ใส่จำนวนที่นับได้จริง** (actual)
   - ระบบแสดง variance:
     - ✅ ตรง (expected = actual)
     - ⚠️ มากกว่า (actual > expected)
     - ❌ น้อยกว่า (actual < expected)
2. ทำซ้ำจนครบทุกรายการ

**Step 3: ส่งผลนับ**
1. ดูสรุป: กี่รายการตรง / กี่รายการไม่ตรง
2. กด `ส่งผลนับ`
3. ระบบ update จำนวนจริง + บันทึก activity log

## 19. Mobile: ค้นหา (Lookup)

**เข้าถึง:** หน้าแรก → `ค้นหา`

### ขั้นตอน
1. **สแกน QR / พิมพ์ค้นหา:**
   - กดปุ่มกล้อง 📷 → สแกน QR Code → ดู Location
   - หรือพิมพ์: Location (เช่น A3/2) หรือ SKU (เช่น L3-8G)
2. **ผลลัพธ์:**
   - ถ้าค้นด้วย Location → แสดงสินค้าทั้งหมดที่ตำแหน่งนั้น
   - ถ้าค้นด้วย SKU → แสดงทุก Location ที่มีสินค้านั้น
3. แต่ละรายการแสดง: ชื่อ, จำนวน, Location
4. กดที่รายการ → ดูรายละเอียด หรือเริ่ม action (ย้าย/ส่งออก)

## 20. Mobile: งานของฉัน (My Tasks)

**เข้าถึง:** หน้าแรก → `งานของฉัน`

### ขั้นตอน
1. ดูรายการ shipment tasks ที่ได้รับมอบหมาย
2. **กรอง:** ทั้งหมด / รอจัด / จัดแล้ว
3. แต่ละ task แสดง:
   - เลข order / shipment
   - ชื่อลูกค้า
   - จำนวนรายการสินค้า
   - ระดับความสำคัญ (สี badge)
   - เวลาที่มอบหมาย
4. **กด `เริ่มจัดของ`** → ไปหน้า Pick โดยอัตโนมัติ
5. ระบบ auto-refresh ทุก 30 วินาที

## Workflow รับสินค้าเข้าคลัง

```
ขั้นตอนเต็ม: PO มาถึง → รับเข้า → ตรวจสอบ → เข้าสต็อก

1. [Desktop] สร้าง/นำเข้า PO ในระบบ
           ↓
2. [Mobile] พนักงานเปิดหน้า "รับสินค้า"
           ↓
3. [Mobile] สแกน PO → เลือกสินค้า → ใส่ Location + จำนวน → ยืนยัน
           ↓
4. [ระบบ] รายการเข้า Staging Queue (สถานะ: pending)
           ↓
5. [Desktop] หัวหน้างาน เปิดหน้า Staging
           ↓
6. [Desktop] ตรวจสอบ → กด "ยืนยัน" → สต็อกเพิ่มขึ้นจริง
           ↓
7. [ระบบ] บันทึก Activity Log อัตโนมัติ
```

```

           ↓

           ↓

           ↓

           ↓

```
